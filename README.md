# Estruturas de Dados — Trabalho Prático

Projeto desenvolvido no âmbito da unidade curricular de **Estruturas de Dados**, com o objetivo de aplicar e implementar estruturas de dados fundamentais na resolução de um problema prático.

## Objetivo
Desenvolver um jogo de labirinto competitivo baseado em turnos, onde os jogadores percorrem um mapa até atingir o objetivo final, lidando com eventos, obstáculos e desafios ao longo do percurso.

## Principais funcionalidades
- Leitura e validação de mapas a partir de ficheiros JSON
- Implementação manual de estruturas de dados (sem Java Collections)
- Jogo por turnos com múltiplos jogadores
- Eventos aleatórios, enigmas e efeitos especiais
- Geração de relatórios finais em formato JSON

## Tecnologias
- Java
- JSON
- JavaDoc

## Contexto académico
Trabalho prático desenvolvido no ano letivo 2024/2025.
